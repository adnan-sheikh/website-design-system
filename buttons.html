Buttons

This section showcases button styles and states that I use very often:

Default State:

  • font-size: 20px
  • text-decoration: none
  • width: 200px
  • padding: 20px
  • background-color: hsla(140, 61%, 56%, 1)
  • border: none
  • border-radius: 20px
  • color: hsla(270, 53%, 37%, 1)
  • cursor: pointer

Hover State:

  • background-color: #FFFFFF
  • border: 2px solid hsla(270, 53%, 37%, 1)
